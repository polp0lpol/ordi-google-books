В репозитории используется файл `google_books_dataset.csv`.  
Он содержит сведения о книгах из Google Books: название, автор, издатель, язык, дата публикации, количество страниц, рейтинг и число отзывов.

## Анализ в Jupyter Notebook

Основной анализ выполнен в ноутбуке `analysis.ipynb`:

- загрузка и первичный осмотр данных;
- приведение типов и обработка пропусков;
- расчёт базовых статистик (распределение рейтингов, количества отзывов, страниц);
- проверка корреляций между числовыми признаками;
- построение нескольких графиков (гистограммы, линейный график по годам, boxplot, heatmap корреляций);
- формулировка основных выводов по датасету.

## Визуализации в Yandex DataLens

Дополнительно построен интерактивный дашборд в Yandex DataLens на основе этого же датасета:

- диаграммы по издателям / языкам;
- динамика среднего рейтинга по годам;
- таблица с агрегированными показателями по издателям (средний рейтинг, число рейтингов, число книг);
- фильтры по языку и году публикации.
